
observable_jupyter_widget
=====================================

Version: |release|

Connect Observable notebooks to the Jupyter kernel


Quickstart
----------

To get started with observable_jupyter_widget, install with pip::

    pip install observable_jupyter_widget

or with conda::

    conda install observable_jupyter_widget
